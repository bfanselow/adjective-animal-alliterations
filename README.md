# adjective-animal-allliterations
Creates random list of "adjective animal" combinations where both start with a specific letter


#### Examples:
```
(venv) wfanselow % ./laughing_lemur.py P -l 10
posted porpoise
pondering platypus
plaintive pony
plantar piranha
pleasurable porcupine
postural penguin
placed prairie-dog
postoperative pika
porcine puma
posterior pheasant
```
